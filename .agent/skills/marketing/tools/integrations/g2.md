# G2

Plataforma de reseñas e investigación de software para compradores B2B. Accede a reseñas, datos de productos, comparaciones con competidores y señales de intención de compra.

## Capacidades

| Integración | Disponible | Notas |
|-------------|-----------|-------|
| API | ✓ | Reviews, Products, Reports, Categories, Tracking |
| MCP | - | No disponible |
| CLI | ✓ | [g2.js](../clis/g2.js) |
| SDK | - | REST API con formato JSON:API |

## Autenticación

- **Tipo**: API Token
- **Header**: `Authorization: Token token={YOUR_API_TOKEN}`
- **Content-Type**: `application/vnd.api+json` (JSON:API)
- **Obtener token**: Portal de administración de G2 > Integrations > API Tokens
- **Documentación**: https://data.g2.com/api/docs

## Operaciones Comunes del Agente

### Listar reseñas (respuestas de encuesta)

```bash
GET https://data.g2.com/api/v1/survey-responses?page[size]=25&page[number]=1

Headers:
  Authorization: Token token={API_TOKEN}
  Content-Type: application/vnd.api+json
```

### Obtener una reseña específica

```bash
GET https://data.g2.com/api/v1/survey-responses/{id}

Headers:
  Authorization: Token token={API_TOKEN}
  Content-Type: application/vnd.api+json
```

### Filtrar reseñas por producto

```bash
GET https://data.g2.com/api/v1/survey-responses?filter[product_id]={product_id}&page[size]=25

Headers:
  Authorization: Token token={API_TOKEN}
  Content-Type: application/vnd.api+json
```

## Patrón de API

G2 sigue la especificación JSON:API (https://jsonapi.org/):
- Las respuestas usan la estructura `data`, `attributes`, `relationships`, `meta`
- Paginación: parámetros de consulta `page[number]` y `page[size]`
- Filtrado: parámetros de consulta `filter[field]=value`
- Las reseñas se devuelven de las más recientes a las más antiguas por defecto (10 por página por defecto)

## Métricas Clave

### Métricas de Reseñas
- `star_rating` - Puntuación general en estrellas
- `title` - Título de la reseña
- `comment_answers` - Respuestas estructuradas de la reseña (gustos, disgustos, recomendaciones)
- `submitted_at` - Fecha de envío de la reseña
- `is_public` - Si la reseña es visible públicamente

### Métricas de Productos
- `name` - Nombre del producto
- `slug` - Slug de URL en G2
- `avg_rating` - Puntuación media en estrellas
- `total_reviews` - Total de reseñas
- `category` - Categoría en G2

### Intención de Compra (Tracking)
- `company_name` - Nombre de la empresa visitante
- `page_visited` - URL de página de G2 visitada
- `visited_at` - Marca de tiempo de la visita
- `activity_type` - Tipo de actividad del comprador

## Parámetros

### Paginación
- `page[number]` - Número de página (por defecto: 1)
- `page[size]` - Elementos por página (por defecto: 10, máximo: 100)

### Filtros de Reseñas
- `filter[product_id]` - Filtrar por ID de producto
- `filter[state]` - Filtrar por estado de la reseña

### Filtros de Tracking
- `filter[start_date]` - Fecha de inicio (YYYY-MM-DD)
- `filter[end_date]` - Fecha de fin (YYYY-MM-DD)

## Cuándo Usar

- Monitorear y analizar reseñas de productos de software
- Rastrear señales de intención de compra de visitantes de G2
- Obtener datos de comparación con competidores para posicionamiento
- Alimentar datos de reseñas en el CRM o automatización de marketing
- Crear contenido de prueba social a partir de reseñas de G2
- Seguimiento de clasificaciones de categorías y ubicaciones en reportes de G2

## Límites de Velocidad

- 10.000 solicitudes por hora por token de API
- Implementar retroceso exponencial en respuestas 429
- Almacenar resultados en caché cuando sea posible para reducir llamadas a la API
