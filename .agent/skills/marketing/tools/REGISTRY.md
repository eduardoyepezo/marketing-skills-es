# Registro de Herramientas de Marketing

Referencia rápida para que los agentes de IA descubran las capacidades de las herramientas y los métodos de integración.

## Cómo Usar Este Registro

1. **Encuentra herramientas por categoría** - Navega por las secciones de abajo para ver las herramientas en cada dominio
2. **Verifica los métodos de integración** - Consulta qué APIs, MCPs, CLIs o SDKs están disponibles
3. **Lee las guías de integración** - Configuración detallada y operaciones comunes en `integrations/`

## Por Categoría

### Analytics

Rastrea el comportamiento de usuarios, mide conversiones y analiza el rendimiento de marketing.

| Herramienta | Ideal para | MCP disponible |
|------|----------|:-------------:|
| **ga4** | Analítica web, ecosistema Google | ✓ |
| **mixpanel** | Analítica de producto, seguimiento de eventos | - |
| **amplitude** | Analítica de producto, análisis de cohortes | - |
| **posthog** | Analítica de código abierto, reproducción de sesiones | - |
| **segment** | Plataforma de datos de clientes, enrutamiento | - |
| **adobe-analytics** | Analítica empresarial | - |
| **plausible** | Analítica enfocada en privacidad | - |

**Recomendación del agente**: Comienza con GA4 si usas el ecosistema Google. Usa Mixpanel o Amplitude para analítica de producto más profunda. Plausible para sitios enfocados en privacidad.

### SEO

Herramientas de optimización para motores de búsqueda para investigación de palabras clave, seguimiento de posiciones y auditorías de sitios.

| Herramienta | Ideal para | Notas |
|------|----------|-------|
| **google-search-console** | Datos de búsqueda gratuitos y autoritativos | Directo de Google |
| **semrush** | Análisis competitivo, investigación de palabras clave | Completo |
| **ahrefs** | Análisis de backlinks, investigación de contenido | El mejor para enlaces |
| **dataforseo** | Seguimiento de SERPs, backlinks, auditorías on-page | API completa |
| **keywords-everywhere** | Investigación rápida de palabras clave, estimaciones de tráfico | Basado en créditos |

**Recomendación del agente**: Google Search Console es esencial (gratuito). Agrega Semrush o Ahrefs para investigación competitiva. DataForSEO para datos de SERP programáticos. Keywords Everywhere para búsquedas rápidas de palabras clave.

## Herramientas CLI

CLIs de Node.js sin dependencias en un solo archivo para herramientas que no incluyen las suyas propias. Consulta [`clis/README.md`](clis/README.md) para instrucciones de instalación y uso.

Todos los CLIs siguen un patrón consistente:
- **Sin dependencias** — Solo Node 18+, usa `fetch` nativo
- **Salida JSON** — enviar a `jq`, guardar en archivo o usar en scripts
- **Autenticación por variable de entorno** — configura `{TOOL}_API_KEY` y listo
- **Comandos consistentes** — `{tool} <recurso> <acción> [opciones]`
