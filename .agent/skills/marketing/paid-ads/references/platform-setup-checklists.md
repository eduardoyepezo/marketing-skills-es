# Listas de Verificación de Configuración de Plataformas

Listas de verificación completas de configuración para las principales plataformas publicitarias.

## Contenidos
- Configuración de Google Ads (Base de la Cuenta, Seguimiento de Conversiones, Integración con Analytics, Configuración de Audiencias, Preparación de Campaña, Extensiones de Anuncio, Protección de Marca)
- Configuración de Meta Ads (Base en Business Manager, Pixel y Seguimiento, Dominio y Eventos Agregados, Configuración de Audiencias, Catálogo, Activos Creativos, Cumplimiento)
- Configuración de LinkedIn Ads (Base en Campaign Manager, Insight Tag y Seguimiento, Configuración de Audiencias, Formularios de Generación de Leads, Document Ads, Activos Creativos, Consideraciones de Presupuesto)
- Configuración de Twitter/X Ads (Base de la Cuenta, Seguimiento, Configuración de Audiencias, Creatividad)
- Configuración de TikTok Ads (Base de la Cuenta, Pixel y Seguimiento, Configuración de Audiencias, Creatividad)
- Lista de Verificación Universal Pre-Lanzamiento

## Configuración de Google Ads

### Base de la Cuenta

- [ ] Cuenta de Google Ads creada y verificada
- [ ] Información de facturación agregada
- [ ] Zona horaria y moneda configuradas correctamente
- [ ] Acceso de cuenta otorgado a los miembros del equipo

### Seguimiento de Conversiones

- [ ] Google tag instalado en todas las páginas
- [ ] Acciones de conversión creadas (compra, lead, registro)
- [ ] Valores de conversión asignados (si aplica)
- [ ] Enhanced conversions habilitado
- [ ] Conversiones de prueba disparándose correctamente
- [ ] Importar conversiones desde GA4 (opcional)

### Integración con Analytics

- [ ] Google Analytics 4 vinculado
- [ ] Auto-tagging habilitado
- [ ] Audiencias de GA4 disponibles en Google Ads
- [ ] Seguimiento entre dominios configurado (si hay múltiples dominios)

### Configuración de Audiencias

- [ ] Tag de remarketing verificado
- [ ] Audiencias de visitantes del sitio web creadas:
  - Todos los visitantes (180 días)
  - Visitantes de páginas clave (precios, demo, funciones)
  - Conversores (para exclusión)
- [ ] Listas de customer match subidas
- [ ] Audiencias similares habilitadas

### Preparación de Campaña

- [ ] Listas de palabras clave negativas creadas:
  - Negativas universales (gratis, empleos, carreras, reseñas, quejas)
  - Negativas de competidores (si es necesario)
  - Términos irrelevantes de la industria
- [ ] Segmentación geográfica configurada (incluir/excluir)
- [ ] Segmentación por idioma configurada
- [ ] Programación de anuncios configurada (si es B2B, horario de oficina)
- [ ] Ajustes de puja por dispositivo considerados

### Extensiones de Anuncio

- [ ] Sitelinks (4-6 páginas relevantes)
- [ ] Callouts (beneficios clave, ofertas)
- [ ] Snippets estructurados (funciones, tipos, servicios)
- [ ] Extensión de llamada (si los leads telefónicos tienen valor)
- [ ] Extensión de formulario de leads (si se usa)
- [ ] Extensiones de precio (si aplica)
- [ ] Extensiones de imagen (donde estén disponibles)

### Protección de Marca

- [ ] Campaña de marca activa (proteger términos de marca)
- [ ] Campañas de competidores consideradas
- [ ] Términos de marca en listas negativas para campañas sin marca

## Configuración de Meta Ads

### Base en Business Manager

- [ ] Business Manager creado
- [ ] Negocio verificado (si se publican ciertos tipos de anuncios)
- [ ] Cuenta publicitaria creada dentro del Business Manager
- [ ] Método de pago agregado
- [ ] Acceso del equipo configurado con roles adecuados

### Pixel y Seguimiento

- [ ] Meta Pixel instalado en todas las páginas
- [ ] Eventos estándar configurados:
  - PageView (automático)
  - ViewContent (páginas de producto/función)
  - Lead (envíos de formularios)
  - Purchase (conversiones)
  - AddToCart (si es e-commerce)
  - InitiateCheckout (si es e-commerce)
- [ ] Conversions API (CAPI) configurada para seguimiento del lado del servidor
- [ ] Puntuación de Event Match Quality > 6
- [ ] Eventos de prueba verificados en Events Manager

### Dominio y Eventos Agregados

- [ ] Dominio verificado en Business Manager
- [ ] Aggregated Event Measurement configurado
- [ ] Los 8 principales eventos priorizados en orden de importancia
- [ ] Eventos web priorizados para el seguimiento de iOS 14+

### Configuración de Audiencias

- [ ] Audiencias personalizadas creadas:
  - Visitantes del sitio web (todos, 30/60/90/180 días)
  - Visitantes de páginas clave
  - Espectadores de video (25%, 50%, 75%, 95%)
  - Personas que interactuaron con la página/Instagram
  - Lista de clientes subida
- [ ] Audiencias lookalike creadas (1%, 1-3%)
- [ ] Audiencias guardadas para segmentaciones frecuentes

### Catálogo (E-commerce)

- [ ] Catálogo de productos conectado
- [ ] Feed de productos actualizándose correctamente
- [ ] Campañas de ventas por catálogo habilitadas
- [ ] Anuncios dinámicos de productos configurados

### Activos Creativos

- [ ] Imágenes en los tamaños correctos:
  - Feed: 1080x1080 (1:1)
  - Stories/Reels: 1080x1920 (9:16)
  - Apaisado: 1200x628 (1.91:1)
- [ ] Videos en los formatos correctos
- [ ] Variaciones de copy de anuncios listas
- [ ] Parámetros UTM en todas las URLs de destino

### Cumplimiento

- [ ] Categorías especiales de anuncios declaradas (si es vivienda, crédito, empleo, política)
- [ ] La landing page cumple con las políticas de Meta
- [ ] Sin contenido prohibido en los anuncios

## Configuración de LinkedIn Ads

### Base en Campaign Manager

- [ ] Cuenta de Campaign Manager creada
- [ ] Company Page conectada
- [ ] Información de facturación agregada
- [ ] Acceso del equipo configurado

### Insight Tag y Seguimiento

- [ ] LinkedIn Insight Tag instalado en todas las páginas
- [ ] Tag verificado y disparándose
- [ ] Seguimiento de conversiones configurado:
  - Conversiones basadas en URL
  - Conversiones basadas en eventos específicos
- [ ] Valores de conversión configurados (si aplica)

### Configuración de Audiencias

- [ ] Matched Audiences creadas:
  - Audiencias de retargeting de sitio web
  - Lista de empresas subida (para ABM)
  - Lista de contactos subida
- [ ] Audiencias lookalike creadas
- [ ] Audiencias guardadas para segmentaciones frecuentes

### Formularios de Generación de Leads (si se usan)

- [ ] Plantillas de formularios de lead gen creadas
- [ ] Campos del formulario seleccionados (minimizar para la conversión)
- [ ] URL de política de privacidad agregada
- [ ] Mensaje de agradecimiento configurado
- [ ] Integración con CRM configurada (o proceso de exportación CSV)

### Document Ads (si se usan)

- [ ] Documentos subidos (PDF, PowerPoint)
- [ ] Acceso configurado (acceso completo o vista previa)
- [ ] Formulario de lead gen conectado

### Activos Creativos

- [ ] Anuncios de imagen única: 1200x627 (1.91:1) o 1080x1080 (1:1)
- [ ] Imágenes de carrusel listas
- [ ] Especificaciones de video cumplidas (si se usa)
- [ ] Copy del anuncio dentro de los límites de caracteres:
  - Texto de introducción: 600 máx., 150 recomendado
  - Titular: 200 máx., 70 recomendado

### Consideraciones de Presupuesto

- [ ] Presupuesto realista para los CPCs de LinkedIn ($8-15+ típico)
- [ ] Tamaño de audiencia validado (50K+ recomendado)
- [ ] Presupuesto diario vs. total decidido
- [ ] Estrategia de puja seleccionada

## Configuración de TikTok Ads

### Base de la Cuenta

- [ ] Cuenta de TikTok Ads Manager creada
- [ ] Verificación de negocio completada
- [ ] Método de pago agregado

### Pixel y Seguimiento

- [ ] TikTok Pixel instalado
- [ ] Eventos configurados (ViewContent, Purchase, etc.)
- [ ] Events API configurada (recomendado)

### Configuración de Audiencias

- [ ] Audiencias personalizadas creadas

### Creatividad

- [ ] Video vertical (9:16) listo
- [ ] Contenido de aspecto nativo (no demasiado pulido)

- [ ] Música/sonidos seleccionados (con licencia si es necesario)

## Lista de Verificación Universal Pre-Lanzamiento

Antes de lanzar cualquier campaña:

- [ ] Seguimiento de conversiones probado con una conversión real
- [ ] La landing page carga rápido (<3 seg)
- [ ] La landing page es compatible con móvil
- [ ] Parámetros UTM funcionando
- [ ] Presupuesto configurado correctamente (diario vs. total)
- [ ] Fechas de inicio/fin correctas
- [ ] La segmentación coincide con la audiencia prevista
- [ ] Creatividad del anuncio aprobada
- [ ] Equipo notificado del lanzamiento
- [ ] Dashboard de reportes listo
